Portal Analizy Wody - Water Analysis Portal

Profesjonalny zestaw narzędzi webowych do przetwarzania, analizy i wizualizacji wyników laboratoryjnych próbek wody (ALS).

🚀 Funkcje Portalu

Portal łączy dwa zaawansowane narzędzia w jednym interfejsie:

Generator ALS (ALS Generator):

Analiza pojedynczych plików Excel z laboratorium ALS.

Automatyczne porównywanie wyników z normami (13 parametrów, w tym metale ciężkie i cyjanek).

Generowanie gotowych do wysłania powiadomień dla klientów w formacie tekstowym i PDF.

Inteligentne oznaczanie przekroczeń kolorami.

Master Timeline:

Łączenie danych z wielu plików Excel w jedną oś czasu.

Wizualizacja trendów na interaktywnych wykresach (Chart.js).

Statystyki zbiorcze: liczba przekroczeń, średnie i maksymalne odchylenia od normy.

Eksport skonsolidowanych danych do formatu .XLSX.

🛠️ Technologie

HTML5 / Tailwind CSS - Nowoczesny i responsywny interfejs.

JavaScript (ES6+) - Silnik obliczeniowy i logika biznesowa.

SheetJS (XLSX) - Zaawansowane przetwarzanie plików arkuszy kalkulacyjnych.

Chart.js - Dynamiczne wykresy trendów.

jsPDF & html2canvas - Generowanie profesjonalnych raportów PDF.

index.html - Główny portal (Launcher) z systemem nawigacji i wyboru języka.
